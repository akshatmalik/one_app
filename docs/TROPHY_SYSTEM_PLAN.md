# Trophy System Plan — 100 Trophies

Comprehensive trophy/award system for Game Analytics. Each trophy has **4 tiers** (Bronze → Silver → Gold → Platinum) with escalating thresholds. All trophies are data-driven — calculated from actual gaming behavior, never random.

**Trophy Score**: Bronze = 10pts, Silver = 25pts, Gold = 50pts, Platinum = 100pts. Total score displayed in header.

**Trophy Room**: Grid UI with earned trophies glowing, locked ones greyed with progress bars. Filter by category, sort by "closest to next tier." Pin 5 favorites to showcase.

**Toast Celebrations**: Confetti toast on earn/upgrade. "You just earned **Gold Completionist**!"

## Implementation Notes

### Data Model

```typescript
interface Trophy {
  id: string;                    // e.g., "century-club"
  name: string;                  // "Century Club"
  icon: string;                  // Emoji
  description: string;           // Short description
  category: TrophyCategory;      // "grind" | "money" | "finisher" | etc.
  isSecret: boolean;             // Hidden until earned
  isMilestone: boolean;          // One-time (no tiers)
  tiers?: TrophyTier[];          // Bronze/Silver/Gold/Platinum thresholds
  milestonePoints?: number;      // Points if milestone (default 50)
}

interface TrophyTier {
  tier: 'bronze' | 'silver' | 'gold' | 'platinum';
  threshold: number | string;    // Numeric threshold or special condition
  description: string;           // Tier-specific description
  points: number;                // 10 / 25 / 50 / 100
}

interface EarnedTrophy {
  trophyId: string;
  tier: 'bronze' | 'silver' | 'gold' | 'platinum' | 'milestone';
  earnedAt: string;              // ISO date
  currentValue: number;          // Current progress value
  isPinned: boolean;             // Showcased in profile
}

type TrophyCategory =
  | 'grind'        // The Grind
  | 'money'        // Money Moves
  | 'finisher'     // The Finisher
  | 'explorer'     // The Explorer
  | 'critic'       // The Critic
  | 'personality'  // The Personality
  | 'first'        // First Blood
  | 'legend'       // The Legend
  | 'secret';      // Secret Trophies
```

### Calculation Function

```typescript
// Master function that evaluates all 100 trophies against current game data
getTrophyProgress(games: Game[]): TrophyProgress[]

// Per-category helpers
getGrindTrophies(games): TrophyProgress[]
getMoneyTrophies(games): TrophyProgress[]
getFinisherTrophies(games): TrophyProgress[]
getExplorerTrophies(games): TrophyProgress[]
getCriticTrophies(games): TrophyProgress[]
getPersonalityTrophies(games): TrophyProgress[]
getFirstBloodTrophies(games): TrophyProgress[]
getLegendTrophies(games, allTrophies): TrophyProgress[]
getSecretTrophies(games): TrophyProgress[]

// Composite
getTrophyScore(trophies: EarnedTrophy[]): number
getPinnedTrophies(trophies: EarnedTrophy[]): EarnedTrophy[]
getClosestToNextTier(trophies: TrophyProgress[]): TrophyProgress[]
```

### Storage

- Earned trophies stored via HybridRepository (localStorage + Firestore)
- Firestore collection: `gameTrophies` with userId field
- Pinned selections stored alongside earned data
- Trophy evaluation runs on every game data change (memoized)

### Components

```
components/TrophyRoom.tsx         — Full trophy grid with filters and sort
components/TrophyCard.tsx         — Individual trophy display (locked/earned/progress)
components/TrophyToast.tsx        — Celebration toast with confetti on earn
components/TrophyShowcase.tsx     — 5-pinned trophies for header/profile
hooks/useTrophies.ts              — Trophy evaluation + earned state management
lib/trophy-definitions.ts         — All 100 trophy definitions
lib/trophy-calculations.ts       — Evaluation logic per trophy
```

### Trophy Score Display

Shown in the Games tab header alongside existing stat cards:
- Trophy icon + total score (e.g., "🏆 847")
- Tap to open Trophy Room
- Mini showcase of 5 pinned trophies below
